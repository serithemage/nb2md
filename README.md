# ipynb파일을 md파일로 변환

노트북 파일이 들어있는 폴더에서 아래 명령어 실행 (여기서는 name.ipynb라는 이름을 지니고 있다고 가정한다)

```sh
jupyter nbconvert name.ipynb --to markdown
```
